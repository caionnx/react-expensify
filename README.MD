# Expensify

A project to learn and pratice React application. See the [live](https://react-expensify-nn.herokuapp.com) demo.

### Scripts

Create bundle as production environment:
```
yarn run build-prod
```
For local development. Accesible from `localhost:8080`:
```
yarn run dev-server
```

### Running the tests

ESLint
```
yarn run js-lint
```
Jest
```
yarn test
yarn test -- --watch
```
